# Experimental Characterization

A more thorough explanation about the synthesis of samples and their experimental characterization (Dynamical Mechanical Analysis (DMA)), can be found in the previously published papers ([Paper 1](https://onlinelibrary.wiley.com/doi/full/10.1002/app.53592), [Paper 2](https://www.mdpi.com/2073-4360/15/22/4434), and [Paper 3](https://onlinelibrary.wiley.com/doi/full/10.1002/eng2.70367)) and the paper linked to this repository. However, a brief overview is provided as follows:

The calibration dataset was obtained from dynamic mechanical analysis (DMA) tension tests on film specimens using a TA Instruments RSA-G2 under linear viscoelastic conditions. Temperature sweeps were performed at `3 °C/min`, and master curves were constructed in TRIOS using time–temperature superposition (TTS) with the reference temperature set to the loss-modulus-peak glass transition temperature, `T_g`. The resulting storage and loss modulus master curves span ten decades of shifted frequency for each nanocomposite system.
